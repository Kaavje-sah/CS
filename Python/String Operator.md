# '+'
```python
s1 = "Hello"
s2 = "World"
print(s1 + s2)  # Output: "HelloWorld"
```

# *
- Replicates the string again and Again
``` python
s1 = "Hello"
s2 = "World"
print(s1 * 3)  # Output: "HelloHelloHello"
print(s2 * 2)  # Output: "WorldWorld"
```

# Slicing
-  Extracts a subset of characters from the string
``` python
s1 = "Hello World"
print(s1[::2]) # Output:  "Hlo ol"

```
